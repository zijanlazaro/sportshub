## Color Coding

### Attendance Rate
- 🟢 Green: ≥ 80% (Excellent)
- 🟡 Yellow: 60-79% (Good)
- 🔴 Red: < 60% (Needs Attention)

### Trends
- ↑ Green: Improving
- → Gray: Stable
- ↓ Red: Declining

### Event Types
- 🟢 Training: Green
- 🔴 Game/Match: Red
- 🔵 Meeting: Blue
- 🟣 Tournament: Purple
- 🟠 Medical Checkup: Orange

## Real-time Features

The dashboard automatically updates when:
1. New medical record is added/updated
2. Training session is created/modified
3. Event is added/changed

**No page refresh needed!**

## Interactive Elements

- **Click on upcoming events** → View full event details
- **Player rows** → View/Edit player information
- **All metrics** → Auto-refresh with real-time data

## Data Time Periods

- **Medical Overview**: All-time + Last 30 days
- **Training Stats**: Last 30 days
- **Performance Trends**: Last 30 days comparison
- **Upcoming Events**: Next 30 days

## Quick Actions

From the overview page you can:
- Navigate to detailed sections (Players, Training, Medical, Schedule)
- View player details and edit information
- See upcoming events and their details
- Monitor team health and performance at a glance

## Tips

1. **Check daily** for real-time team status
2. **Monitor trends** to catch issues early
3. **Review follow-ups** to ensure player care
4. **Track attendance** to maintain team discipline
5. **Plan ahead** with upcoming events visibility
