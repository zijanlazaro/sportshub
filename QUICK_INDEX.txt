   → Run docs/DATABASE_SCHEMA.sql
   → Run docs/RLS_POLICIES.sql

2. Configure Environment
   → cp .env.example .env
   → Add Supabase credentials

3. Run Locally
   → npm install
   → npm run dev
   → Visit http://localhost:3000
