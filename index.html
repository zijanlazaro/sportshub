Athlete Management Made Simple

Sports Hub is a comprehensive thesis-level athlete management and analytics system designed for sports teams, coaches, and medical staff to track, manage, and analyze athlete data efficiently.

Get Started Learn More

Key Features

Player Management

Create and manage comprehensive player profiles including personal data, jersey numbers, positions, and physical metrics.

Performance Analytics

Track player statistics including games played, points scored, assists, and performance trends over time.

Medical Records

Maintain detailed injury history, medical notes, recovery status, and clearance documentation for each player.

Training Attendance

Create training sessions and track attendance with status options: Present, Absent, or Excused.

Event Scheduling

Schedule and manage games, training sessions, and team events with status tracking and calendar views.

Role-Based Access

Role-based security with Admin, Coach, Medical Staff, and Player roles using Supabase Row Level Security.

Tech Stack

Frontend

HTML5, JavaScript ES6+, Tailwind CSS

Database

Supabase (PostgreSQL, Real-time)

Authentication

Supabase Auth, RLS Policies

Hosting

Netlify (Static Site)
